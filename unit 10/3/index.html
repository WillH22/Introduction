  1. What git command creates a branch?

    2. git checkout -b (branch name)

  2. What is the difference between a fast-forward and recursive merge?

    3. Fast forward merge is when git can easily tell when the commits happened but only if it is chronological whereas Recursive merge is usually used when a fast forward merge won't work

  3. What git command changes to another branch?

    4. git checkout (branch name)

  4. What git command deletes a branch?

    5. git branch -d (branch name)

  5. How do merge conflicts happen?

    6. When changes are committed to the same file from two different branches
