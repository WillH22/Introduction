  1. Create a folder called learn_git_again

    2. rmdir learn_git_again

  2. cd into the learn_git_again folder

    3. cd learn_git_again

  3. Create a file called third.txt

    4. touch third.txt

  4. Initialize an empty git repository

    5. git init

  5. Add third.txt to the staging area.

    6. git add third.txt

  6. Commit with the message "adding third.txt"

    7. git commit -m "adding third.txt"

  7. Check out your commit with git log

    8. git log

  8. Create another file called fourth.txt

    9. touch fourth.txt

  9. Add fourth.txt to the staging are

    10. git add fourth.txt

  10. Commit with the message "adding fourth.txt"

    11. git commit -m "adding fourth.txt"

  11. Remove the third.txt file

    12. rm third.txt

  12. Add this change to the staging area

    13. git rm third.txt

  13. Commit with the message "removing third.txt"

    14. git commit -m "removing third.txt"

  14. Check out your commits using git log

    15. git log

  15. Change your global setting core.pager=cat -you can read more about that [here]

    16. git config --global core.pager"cat"

  16. Write the command to list all of the global configurations for git on your machine. You can type git config --global to find out how to do this

    17. git config -global -list
