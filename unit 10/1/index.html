Part I

  1. Make a directory called first

    2. mkdir first

  2. Change directory to the first folder

    3. cd first

  3. Create a file called person.txt

    4. touch person.txt

  4. Change the name of person.txt to another.txt

    5. mv person.txt another.txt

  5. make a copy of the another.txt file and call it copy.txt

    6. cp another.txt copy.txt

  6. remove the copy.txt file

    7. rm copy.txt

  7. make a copy of the first folder and call it second

    8. cp -r first second

  8. delete the second folder

    9. rmdir second or you could also do rm -rf second (incase it has any files inside)

Part II

  1. What does the man command do? type in man rm. How do you scroll and get out?

    2. Man command lets you see a manual on the command you're using/want to use. You scroll using enter and can exit by pressing Q

  2. Look at the man page for ls. What does the -l flag do? What does the -a flag do?

    3. -l uses a long listing format, -a doesn't ignore entries starting with .

  3. How do you jump between words in the terminal?

    4. option/alt + left/right

  4. How do you get to the end of a line in terminal?

    5. Ctrl + e

  5. How do you move your cursor to the beginning in the terminal?

    6. Ctrl + a

  6. How do you delete a word (without pressing backspace multiple times) in terminal?

    7. Ctrl + w

  7. What is the difference between a terminal and shell?

    8. Shell processes commands and returns output and a terminal runs a shell

  8. What is an absolute path?

    9. Its a specified path starting from the root directory

  9. What is a relative path?

    10. Its the path related to the pwd

  10. What is a flag? Give three examples of flags you have used.

    11. They change and enhance commands, some of the flags I've uses would be -b, -d, and -m

  11. What do the r and f flags do with the rm command?

    12. f attempts to remove the files without prompting for confirmation. r is equivalent to R which attempts to remove the file hierarchy rooted in each file argument.
