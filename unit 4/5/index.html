Objects Exercise

let programming = {
languages: ["JavaScript", "Python", "Ruby"],
isChallenge: true,
isRewarding: true,
difficulty: 8,
jokes: "https://bit.ly/2ysFran"
};
  1. Write the command to add the language "Go" to the end of the languages array.

    2. programming.languages.push("Go")

  2. Change the difficulty to the value 7

    3. programming.difficulty = 7

  3. Using the delete keyword, write the command to remove the jokes key from the programming object.

    4. delete programming.jokes

  4. Write the command to add a new key called isFun and a value of true to the programming object

    5. programming.isFun = true
