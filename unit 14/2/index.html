Quiz

  • What is the difference between var and let?

    • Variables declared by let are only available inside the block where they're defined, let can also be reassigned whereas var can be redeclared and reassigned

  • What is the difference between var and const?

    • const stays the same basically, it cannot be redeclared or reassigned, where you can with var.

  • What is the difference between let and const?
    • let can be reassigned but const can't.
  • What is hoiusing?

    • its a mechanism where variables and function declarations are moved to the top of their scope before code execution
