Part 1 - Conceptual

  • What is a CSS Selector? Please give an example

    • A selector is something we would like to style, an example is of a selector will be in bold letters -

    body {
    background-color: blue;
    }
  • What is a CSS Declaration? Please give an example.

    • Using the example above, everything inside the curly braces is the declaration.

  • What is a CSS property? Please give an example.

    • Using the example above, everything to the left of the colon is property.

  • What is a CSS value? Please give an example.

    • Using the example above, everything to the right of the colon followed by a semicolon is the value.

  • What are the three ways to include CSS?

    • 3 ways to include CSS is by using <style> in HTML as an element, there's also external stylesheet <link>, and lastly is inline which shouldn't be used

  • What is the difference between rgb and rgba?

    • rgb and rgba are basically the same, the only difference is rgba, the a is basically for opacity whereas in rgb you can't set opacity.

  • What is an em?

    • an em is just a font size

  • What are the differences between em and rem?

    • em is relative to the font size and rem is relative to the html size
